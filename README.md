# Computer Vision

Dự án thực hành môn Xử lý ảnh (Computer Vision).

## Cấu trúc thư mục
- `Lab1.ipynb`: Notebook bài tập/thực hành.
- `in.jpg`: Ảnh đầu vào.
- `.venv/`: Môi trường ảo (Virtual Environment).

## Cài đặt & Sử dụng
1. Kích hoạt môi trường ảo (trên Windows):
   ```powershell
   .\.venv\Scripts\activate
   ```
2. Cài đặt các thư viện cần thiết (nếu chưa cài):
   ```powershell
   pip install -r requirements.txt
   ```
3. Mở Jupyter Notebook để xem và chạy code:
   ```powershell
   jupyter notebook
   ```
